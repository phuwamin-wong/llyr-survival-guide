# Pediatrics Advance Life Support

## What to ask back when being notified?

* Sex, Age, Diagnosis และ Underlying diseases
* เด็กได้รับ Management เบื้องต้นอย่างไรบ้าง
* Vital sign: BP, HR, RR, SpO<sub>2</sub>, Body Temp, EKG
* รวบรวมข้อมูลทั้งหมดแจ้ง Intern/Resident/Staff และรีบไปประเมินผู้ป่วยที่เตียงทันที

## Initial Impression (PAT: Pediatrics Assessment Triangle)

การประเมินแรกพบ (Quick look) ด้วยสายตาโดยไม่ต้องจับตัวเด็ก:

* **A** (Appearance): Abnormal / Absent crying or speech / Decreased response / Floppy / Not moving
* **B** (Work of Breathing): Nasal flaring / Chest retraction / Decreased respiratory effort / Noisy breathing
* **C** (Circulation to Skin): Pallor / Mottling / Cyanosis

**Vital Signs:** ค่าปกติแตกต่างตามช่วงอายุ แนะนำให้เปิดตารางอ้างอิงเสมอ (เฝ้าระวัง BP, HR, RR)

**3 Ways Possible (เมื่อประเมิน Pulse และ Breathing แล้ว):**

1. No breathing + No pulse $$→$$ Start CPR ทันที
2. No breathing + Pulse present $$→$$ Rescue breathing (Maintain airway + Ventilation/Oxygenation)
3. Breathing + Pulse present $$→$$ รอดตายเฉพาะหน้า ให้ไปประเมินแบบละเอียดตาม Primary assessment (ABCDE)
{% endhint %}

## Initial Management

* เมื่อทราบแล้วว่าต้อง Run algorithm ไหน ให้ Start CPR หรือจัดการ Definite airway ได้ทันทีหลังจากมีคนแจ้งทีม และ Notify Intern, Resident, Staff แล้ว
* High-quality CPR: 5 components
  * Depth
    * 1/3 AP diameter
    * Infant: 4 cm
    * Children 5 cm
  * Minimize interruption
  * Fully recoil
  * Avoid excess ventilation
    * 1 rescuer → 30:2
    * 2 rescuers → 15:2
    * Advance airway: บีบ Ambu bag ทุก 2-3 วินาที โดยไม่ต้องหยุด CPR
* Check pulse <10 sec, every 2 min
* <mark style="color:purple;">Ventilation more important for infant</mark>

{% hint style="info" %}
ในเด็ก ภาวะ Arrest มักเกิดจาก Hypoxia มากกว่าปัญหาหัวใจ การทำ Ventilation จึงมีความสำคัญมากๆ ห้ามละเลยเด็ดขาด
{% endhint %}

### Airway Management

**Open airway**

* Head tilt – chin lift
* Jaw thrust (กรณีไม่สามารถ out C-spine injury)
* Bag-Mask Ventilation: ใช้ Self-inflating bag with reservoir, เปิด Oxygen 10-15 L/min
  * ทารกและเด็กเล็ก ใช้ถุงขนาด 450-500 mL
  * เด็กโต ใช้ถุงขนาด 1,000 mL
* Respiratory rate 20-30/min

{% hint style="info" %}
Adequate Bag-Mask ventilation → Outcome ไม่ด้อยกว่า Endotracheal intubation
{% endhint %}

**Endotracheal Intubation**

* แนะนำให้ใช้ Cuffed ETT เป็นหลัก (Cuff inflation pressure < 20-25 cmH2O)
* ไม่แนะนำให้ทำ Routine cricoid pressure
* ขนาด Cuffed ETT: (Age / 4) + 3.5
* ความลึก ETT (Depth): (Age / 2) + 12 _(หรือจำง่ายๆ ว่า เอาเบอร์ Tube x 3)_

#### Complete Airway Obstruction with Consciousness (Foreign Body Airway Obstruction)

* Infant (< 1 ปี): 5 Back blows + 5 Chest thrusts
* Child ($$≥$$ 1 ปี): Abdominal thrusts (Heimlich maneuver)

{% hint style="info" %}
⚠️ คำเตือน: ห้ามทำ Blind finger sweep เด็ดขาด ให้หยิบออกต่อเมื่อมองเห็นสิ่งแปลกปลอมชัดเจนเท่านั้น
{% endhint %}

* หากระหว่างทำ คนไข้หมดสติ (Unresponsive) $$→$$ ให้จับนอนราบ และ Start CPR ทันที (เริ่มด้วยการปั๊มหน้าอก ไม่ต้องคลำชีพจร)

## Hypoglycemia Management

เด็กที่ป่วยหนักหรือ Arrest มักมีปัญหาน้ำตาลต่ำร่วมด้วยเสมอ

* Neonate (< 45 mg/dL): ให้ 10%DW 2 ml/kg (IV slowly push)
* Infant / Child / Adolescent (< 60 mg/dL):
  * จำง่ายๆ ด้วย "Rule of 50" (เอาเปอร์เซ็นต์น้ำตาล x ปริมาตร = 50)
  * ให้ 10%DW 5 ml/kg (IV) หรือ
  * ให้ 25%DW 2 ml/kg (IV)

{% hint style="info" %}
หลีกเลี่ยงการใช้ 50%DW ในเด็กเพื่อป้องกันเส้นเลือดอักเสบ (Vein sclerosis) ยกเว้นในวัยรุ่นตัวโตหรือกรณีฉุกเฉินที่ไม่มีทางเลือก
{% endhint %}
