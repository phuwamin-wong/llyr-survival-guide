# Post-ND round

## **Postpartum care : The important 10 B’s**&#x20;

* Blues : ประเมินภาวะซึมเศร้าหลังคลอด
* Breast : น้ำนมน้อย คัดตึงเต้านม ติดเชื้อ Latch on
* Belly : Fundal height แผลผ่าคลอดหน้าท้อง&#x20;
* Bottom : Episiotomy & Perineum
* Bowel : อืดท้อง ถ่ายไม่ออก
* Bladder : ฉี่ไม่ออก
* Bleeding : Lochia
* Boinking  : Advice avoid SI
* Baby :  Daily care, Latch on, Sleep
* Birth control : Contraception หลังจากครรภ์นี้

## **Patient evaluation “Post ND Round”**

* ประเมิน SOAP ตามปกติได้เลย + ถามเรื่อง 10 B

**Subjective**

* ปวดแผล&#x20;
* ambulate, ฉี่, ขับถ่าย&#x20;
* เลือดออก/สีน้ำคาวปลา+ปริมาณกี่แผ่นต่อวัน&#x20;
* คัดตึงเต้านม น้ำนมไหลน้อย เจ็บเต้านมตอนเข้าเต้า ถาม Latch on, breastfeeding ทุกวัน&#x20;
* Postpartum blue&#x20;
* ถามเรื่องลูก ตัวเหลือง/ถ่าย/ฉี่/หลับ&#x20;

**Objective ⚠️ควรมีบุคคลที่สามอยู่ด้วย**

* V/S ดูไข้&#x20;
* HR Breast >>> engorge? น้ำนมมายัง=รีดดูว่ามีน้ำนมออกมาจากหัวนมมั้ย, บวมแดงร้อน
* Heart\&Lungs&#x20;
* Abdomen : ท้องอืดเยอะ ไม่ถ่าย
* Uterus: Fundal height (involute ดีมั้ย), tone >>> วาดรูป fundal height ไว้เลย&#x20;
* Episiotomy: แผลแยก แผลบวม hematoma (วาดรูป)&#x20;
* Lochia rubra/serosa/alba (แค่ถามก็ได้ ไม่ต้องเปิดดู)

**Assessment**

\#P1001, GA 39+2 wk with \_\_\_ (Risk ทั้งหมด) s/p ND or C/S day\_\_\_&#x20;

\#problem ใหม่ที่เจอตอนราวน์&#x20;

**Plan**

Ambulate แก้ปัญหาที่เจอตอนราวร์ ซึกมักจะได้แก่

* ฉี่ไม่ออก -> u/s bladder, single catheter, UA &#x20;
* น้ำนมไม่ไหล,น้อย -> พบแพทย์แผนไทย, Domperidone 1x3 po pc
* ปวดแผล hematoma  -> pain control +- ATB&#x20;
* ท้องอืด -> Promote ambulation, Simethicone 1x3 po pc
* ถ่ายไม่ออก -> Observe, Lactulose 30 ml po hs
* Promote breastfeeding
* Advice Advice Birth control method -> ATR?, Implanon?, POP? \*\*\*
* ดูแลต่อเนื่องตาม risk ที่มีตอนท้อง e.g. preeclampsia, GDM&#x20;
* D/C post ND 2 days, post C/S 3 days + F/U 6 wk PP clinic&#x20;
* Advice เรื่องต่างๆที่ให้ทำต่อเนื่อง,เรื่อง SI, เรื่อง breastfeeding, เรื่องอาหารกินได้เลย etc.
