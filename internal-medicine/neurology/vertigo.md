# Vertigo

## Definition

Vertigo ความรู้สึกหลอนของการเคลื่อนไหว (Illusion of movement) โดยผู้ป่วยจะรู้สึกว่าตนเองหรือสิ่งแวดล้อมรอบตัวกำลังเคลื่อนที่ ทั้งที่ความเป็นจริงไม่ได้มีการเคลื่อนที่เกิดขึ้น ลักษณะอาการที่พบได้บ่อยมักถูกอธิบายด้วยคำว่า "**บ้านหมุน**" (Spinning) หรือ "**รู้สึกโคลงเคลงเหมือนอยู่บนเรือ**" (Swaying/Rocking)

## Approach

### Rule-out red flag signs

{% hint style="danger" %}
ต้อง Clarify ให้ได้ว่าผู้ป่วยมีอาการเป็น true vertigo หรือเป็นอาการอื่น ๆ ซึ่ง life-threatening
{% endhint %}

1. **True Vertigo (บ้านหมุน/โคลงเคลง)**: ชี้เป้าไปที่ความผิดปกติของระบบทรงตัว (Vestibular system) ซึ่งอาจเป็นได้ทั้ง Peripheral (เช่น BPPV, Meniere's) หรือ Central (เช่น Cerebellar infarction)
2. **Pre-syncope / Syncope (หน้ามืด/วูบ/หวิวเหมือนจะหมดสติ)**: ผู้ป่วยรู้สึกตาพร่ามัว หน้ามืดเวลาเปลี่ยนท่าทาง ต้อง Rule-out ภาวะทาง Cardiovascular เสมอ (เช่น Arrhythmia, Orthostatic hypotension, หรือ Structural heart disease)
3. **Diplopia (เห็นภาพซ้อน):** ผู้ป่วยบางราย (โดยเฉพาะผู้สูงอายุ) อาจอธิบายอาการมองเห็นภาพซ้อนว่า "เวียนหัว" ซึ่งเป็นหนึ่งในอาการแสดงของ Transient Ischemic Attack (TIA) หรือ Stroke ในระบบไหลเวียนโลหิตส่วนหลัง (Posterior circulation)
4. **Disequilibrium (เดินเซ/ทรงตัวไม่อยู่):** รู้สึกโคลงเคลงเฉพาะเวลาเดิน มักเกิดจากปัญหาทางระบบประสาท (เช่น Cerebellar ataxia, Peripheral neuropathy)

{% hint style="warning" %}
เมื่อคนไข้บอกว่าเวียนหัว ให้ถามเจาะจงเสมอว่า _"ที่ว่าเวียนหัว คือเห็นสิ่งแวดล้อมหมุนรอบตัว หรือรู้สึกหน้ามืดเหมือนจะเป็นลม หรือเดินเซทรงตัวไม่ได้?"_
{% endhint %}

### History and Physical Examination

หากซักได้ว่าเป็น True vertigo ให้เน้นไปที่ประวัติเพื่อแยก Central และ Peripheral Vertigo

**Clinical Signs and Symptoms suggestive of Central Vertigo**

หากพบอาการหรือเครื่องหมายต่อไปนี้ ให้สงสัยว่าสาเหตุมาจาก Central lesion (ซึ่งมักเป็น Life-threatening conditions) ทันที:

1. Focal Neurodeficits (ความผิดปกติทางระบบประสาทเฉพาะที่): \* Weakness (แขนขาอ่อนแรง), Facial weakness (หน้าเบี้ยว), Sensory change (ชาครึ่งซีก)
   * Dysmetria (กะระยะผิดพลาด ทั้งแบบ Limb หรือ Truncal ataxia)
   * Diplopia (เห็นภาพซ้อน), Dysarthria (พูดพูดไม่ชัด/อ้อแอ้), Dysphagia (กลืนลำบาก), และ Dysphonia (เสียงแหบ)
2. Spontaneous Vertical Nystagmus: พบการตากระตุกในแนวดิ่งขณะมองตรง (Primary gaze)
   * _ข้อยกเว้น:_ หากกระตุกแนวดิ่งร่วมกับมีการบิดหมุน (Vertical + Torsional/Rotatory component) มักจะหมายถึงภาวะ BPPV (ซึ่งกระตุ้นให้เกิดด้วยท่า Dix-Hallpike test)
3. Significant and Persistent Headache: เวียนศีรษะร่วมกับปวดศีรษะรุนแรงและต่อเนื่อง $$\rightarrow$$ ให้นึกถึง Cerebellar hemorrhage / Infarction
4. Neck Pain: เวียนศีรษะร่วมกับปวดคอ $$\rightarrow$$ ให้นึกถึง Vertebral artery dissection

* Central Cause: ผู้ป่วยที่เวียนศีรษะรุนแรงจน ยืนเองไม่ได้ ทรงตัวไม่อยู่ และต้องมีคนช่วยพยุง ให้ถือว่าเป็น _Central cause of vertigo ไปก่อนเสมอ_ จนกว่าจะพิสูจน์ได้ว่าเป็นอย่างอื่น (เช่นทำ CT Brain/MRI แล้วปกติ)

### Key Bedside Test

#### Dix-Hallpike Test

เป็นการตรวจเพื่อวินิจฉัยภาวะ Benign Paroxysmal Positional Vertigo (BPPV) โดยเฉพาะ Posterior canal

**📌 ข้อบ่งชี้ในการทำ (Indications):** พิจารณาทำในผู้ป่วยที่มีอาการเข้าได้กับ BPPV **ครบทั้ง 3 ข้อ** ดังนี้:

1. Acute Vertigo ที่มีระยะเวลาสั้นๆ (Lasting less than 2 minutes ต่อการเวียน 1 ครั้ง)
2. อาการเวียนศีรษะถูกกระตุ้น เมื่อมีการเปลี่ยนแปลงท่าทางของศีรษะ (Position-evoked)
3. ไม่มี Spontaneous หรือ Gaze-evoked nystagmus ขณะนั่งมองตรงนิ่งๆ _(หากประวัติไม่เข้า Criteria เหล่านี้ หรือมี Nystagmus ตลอดเวลา ให้พิจารณาข้ามไปทำ HINTS-Plus test แทน)_

**หากไม่เข้า Criteria ทั้งหมดนี้ ให้พิจารณาทำ** [**HINTS-Plus test**](vertigo.md#hints-plus-test-head-impulse-nystagmus-test-of-skew-test) **(see below)**

🛠️ วิธีการตรวจ (Steps):

1. จัดเตียงให้ราบ โดยคำนวณตำแหน่งให้ศีรษะคนไข้สามารถห้อยตกลงมานอกเตียงได้พอดี
2. ให้ผู้ป่วยนั่งตรง และให้หันหน้า (Rotate คอ) ไปยังด้านที่จะทดสอบ (เช่น ด้านขวา) 45 องศา
3. ให้ผู้ป่วยล้มตัวลงนอนอย่างรวดเร็ว โดยให้ศีรษะห้อยต่ำลงจากขอบเตียง (Extend) ประมาณ 20 องศา
4. ลืมตาค้างไว้เบิกกว้าง ๆ **รอสังเกตอาการประมาณ 15-30 วินาที** หากลูกตาหมุนไปด้านไหน แปลผลได้ว่ามี Posterior canal BPPV ในหูข้างนั้น

{% embed url="https://www.youtube.com/watch?v=EdzHlD16iOI" %}

{% embed url="https://www.youtube.com/watch?v=7ePecb9azS4" %}

{% hint style="info" %}
✅ การแปลผล Positive สำหรับ Posterior Canal BPPV:

* Latency: อาการตากระตุก (Nystagmus) มักไม่เกิดทันที แต่จะเริ่มเกิดภายใน 15 วินาทีแรก
* Character: ลักษณะตากระตุกต้องเป็นแบบ **Vertical upward and rotatory nystagmus** (ตากระตุกขึ้นบนและมีการบิดหมุน)
* Duration: อาการจะค่อย ๆ เป็นมากขึ้น แล้วค่อย ๆ เบาลงจนหายไปเองภายใน 15-30 วินาที (Crescendo-decrescendo pattern)
{% endhint %}

#### HINTS-Plus test (<mark style="background-color:pink;">H</mark>ead <mark style="background-color:pink;">i</mark>mpulse, <mark style="background-color:pink;">N</mark>ystagmus, <mark style="background-color:pink;">T</mark>est of <mark style="background-color:pink;">S</mark>kew) test Plus Hearing Loss

เป็นการตรวจเพื่อแยก Central Vertigo ออกจาก Peripheral Vertigo ในผู้ป่วยที่มีภาวะ Acute Vestibular Syndrome (AVS)

{% hint style="danger" %}
การตรวจ HINTS จะมีประโยชน์และแม่นยำก็ต่อเมื่อ ทำในผู้ป่วยที่มีข้อบ่งชี้ถูกต้องเท่านั้น!
{% endhint %}

**📌 ข้อบ่งชี้ในการทำ (Indications):**

* ทำในผู้ป่วยที่มีอาการเวียนศีรษะต่อเนื่องยาวนานเป็นชั่วโมงหรือเป็นวัน (Persistent continuous ongoing vertigo for hours/days) **และ**
* **ต้องมี** Spontaneous หรือ Gaze-evoked nystagmus ให้เห็นชัดเจน

**🛠️ วิธีการตรวจและแปลผล (ตามลำดับดังนี้):**

{% stepper %}
{% step %}
### Nystagmus

{% hint style="danger" %}
<mark style="color:$danger;">**Only patients with persistent ongoing vertigo and spontaneous or gaze-evoked nystagmus should undergo HINTS testing.**</mark>
{% endhint %}

**วิธีตรวจ:** ให้ผู้ป่วยมองลอยๆ มองตรงไปข้างหน้าเพื่อหา _Spontaneous nystagmus_ จากนั้นให้ตรวจเหมือนการประเมิน EOM (ให้มองซ้ายสุด ขวาสุด บนสุด ล่างสุด) เพื่อหา _Gaze-evoked nystagmus_

**การรายงานผล:** ต้องระบุเสมอว่า

1. เป็นแบบ Spontaneous หรือ Gaze-evoked
2. ทิศทางเป็นอย่างไร (Horizontal, Vertical, Torsional)
3. Fast phase และ Slow phase ตีไปทางด้านไหน

**การแปลผล:** หากทิศทางของ Fast phase เปลี่ยนไปตามทิศทางการมอง (Direction-changing) $$\rightarrow$$ สงสัย Central lesion
{% endstep %}

{% step %}
### Test of Skew

* **วิธีตรวจ:** ใช้วิธีการ Alternate Cover Test
  1. ให้ผู้ป่วยจ้องมองที่จมูกของผู้ตรวจนิ่ง ๆ
  2. ปิดตาข้างหนึ่งของผู้ป่วยไว้ แล้วสลับมาปิดตาอีกข้างหนึ่งของผู้ป่วยอย่างรวดเร็ว
* **การแปลผล:** ขณะที่สลับมือเปิดตา หากพบว่าลูกตามีการขยับปรับตำแหน่ง (Corrective movement) ขึ้นลงในแนวดิ่ง หรือแนวทแยง (Vertical/Diagonal deviation) แปลผลว่า Positive $$\rightarrow$$ สงสัย **Central lesion**

{% embed url="https://www.youtube.com/watch?v=zgqCXef-qPs" %}
{% endstep %}

### Head Impulse Test (ประเมิน Vestibulo-ocular reflex)

วิธีตรวจ:

1. ใช้สองมือจับศีรษะผู้ป่วยให้มั่นคง
2. ให้ผู้ป่วยจ้องมองที่จมูกของผู้ตรวจตลอดเวลา และบอกผู้ป่วยว่า "ไม่ต้องเกร็งคอ ปล่อยตามสบาย"
3. หมุนคอผู้ป่วยประมาณ 20 องศาจาก Midline อย่างรวดเร็วแต่นุ่มนวล (โดยให้ผู้ป่วยยังคงจ้องจมูกผู้ตรวจอยู่)

การแปลผล:

* Abnormal VOR: หากตอนหมุนหน้า ลูกตาของผู้ป่วยขยับตามศีรษะไป (หลุดจากการโฟกัสที่จมูก) แล้วเกิดการกระตุกตากลับมาเพื่อโฟกัสที่จมูกใหม่ เรียกว่าเกิด **"Catch-up Saccade"** $$\rightarrow$$ แบบนี้มักแปลว่าเส้นประสาทหูมีปัญหา ซึ่งเข้าได้กับ **Peripheral lesion** (โดย Saccade มักเกิดในทิศเดียวกับ Fast phase ของ Spontaneous nystagmus ที่ตรวจพบตอนแรก)
* **Normal VOR:** หากผู้ป่วยสามารถ Fixate สายตาไว้ที่จมูกผู้ตรวจได้ตลอดเวลาโดยไม่มีลูกตากระตุกหลุดเลย $$\rightarrow$$ ให้ระวังไว้ว่านี่คือ **Central lesion!!** (เพราะรอยโรคในสมอง/Stroke ส่วนใหญ่มักไม่ได้ทำลายวงจร VOR)

{% hint style="danger" %}
## Normal Head Impulse Test = Centreal Lesion

ระวัง! รอยโรคในสมอง/Stroke ส่วนใหญ่มักไม่ได้ทำลายวงจร VOR
{% endhint %}
{% endstep %}

{% step %}
### the Plus test

คำว่า Plus ที่เพิ่มเข้ามา หมายถึงการประเมินการได้ยิน (Hearing test) โดยการใช้นิ้วถูหรือทำเสียงกระซิบข้างหู หากพบว่ามี **New onset unilateral hearing loss** (หูดับข้างเดียวเฉียบพลัน) ให้สงสัยว่ารอยโรคอาจเป็น **AICA Infarction** ซึ่งเป็น Central Vertigo ได้เช่นกัน
{% endstep %}

## Management

{% hint style="info" %}
หากสงสัย Central lesion ⇒ Investigation and treatment as stroke
{% endhint %}

### BPPV (Benign paroxysmal positional vertigo)

**First-line treatment is Epley's Maneuver (**&#x63;analith repositioning maneuver&#x73;**)**
