# Lower Gastrointestinal Hemorrhage

## Definition

* Bleeding จาก GI tract ที่อยู่ต่ำกว่า Ligament of Treitz

## What to ask back when being notified

1. **Vital Signs:** ถามหา signs ของ Hemodynamic instability ทันที (เช่น Tachycardia, Hypotension)
2. **Consciousness:** ผู้ป่วยยังตื่นรู้ตัวดีหรือไม่
3. **ลักษณะและปริมาณเลือด:** ออกมาประมาณกี่ซีซี? เป็นเลือดสดปนลิ่มเลือด หรือเป็น melena
4. Current Medications: \* ยาที่ทำให้เกิดภาวะ Pseudo-melena: Bismuth, Ferrous fumarate (ธาตุเหล็ก)
   * ยาที่เพิ่มความเสี่ยง Bleeding: ยาแก้ปวดกลุ่ม NSAIDs, ยาต้านเกล็ดเลือด (ASA, Clopidogrel), และยาต้านการแข็งตัวของเลือด (Enoxaparin, DOACs, Warfarin)

{% hint style="warning" %}
อย่าลืมประเมิน Massive UGIH ที่อาจมา present ด้วย LGIH ได้!!
{% endhint %}

## Patient Evaluation

ทำการประเมินในลักษณะเดียวกับ Upper GI hemorrhage นั่นคือเริ่มที่ Airway, Breathing, Circulation ก่อน หากมีปัญหาให้แก่ไปตามเรื่องตามราว

{% hint style="info" %}
ควรแยกว่าเป็น bleeding per rectum หรือเหนือกว่า rectum ด้วยการ proctoscope ในทุกเคสหากเป็นไปได้
{% endhint %}

การ resuscitation ทำคล้ายกับ Upper GI hemorrhage

## Management

* หา Cause หากเป็นไปได้ เพราะถ้าเป็น Hemorrhoid หรือ Anal fissure สามารถรักษาได้ด้วยการให้ยาหากไม่ massive
* กรณี bleeding above proctoscope หากไม่ stable ต้องพิจารณา consult ศัลยกรรม รวมทั้งอาจต้องพิจารณา imaging เช่น CTA และ/หรือ colonoscopy
  * กรณีมี bleeding จน hemodynamic unstable --> angiography + embolization
