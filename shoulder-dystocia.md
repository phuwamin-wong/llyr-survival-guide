# Shoulder Dystocia

{% hint style="info" %}
Unpredictable emergency condition

Previous dystocia --> recurrent rate 10%

Turtle neck sign
{% endhint %}

## Definition

* anterior fetal shoulder wedged behind the symphysis pubis and fail to deliver with maternal pushing and gentle axial traction by the provider&#x20;
* **head-to-body delivery time > 60 seconds** be used to define shoulder dystocia/ maneuvers are required to free the shoulder

## What to ask back when being Notified?

* หัวออกมานานเท่าไหร่ + วิ่ง + ตามพี่/อจ. Stat

**สิ่งที่ไม่ควรทำ**

* Don’t Fundal pressure
* Excessive traction, Head rotation

## Management

* Call for help + Keep calm สำคัญที่สุด!! (ตามอาจารย์หรือพี่เด้นให้เร็วที่สุด)
* Generous Episiotomy กว้างพอหรือยัง&#x20;
* Emptying bladder (retained foley catheter)&#x20;
* Monitor FHR, suction, Notify Ped emer

### Maneuvers

1. McRoberts maneuver ร่วมกับ Suprapubic 🌟→  ปลดขาของแม่ ยกขึ้น งอเข่าพับหาท้อง ให้ hip hyperflexion + abduction suprapubic pressure กดให้ไหล่ด้านหน้า หลุดออกจาก pubic symphysis ให้กดตอน แม่เบ่ง + เราดึงเด็กออกมา : เน้น Synchronize กัน&#x20;
2. Posterior arm release → คลอดไหล่หลังก่อน ใช้มือปาดแขนหลังตรง ศอกผ่านทางหน้าออกมา แล้วคลอดแขนข้าง นั้น จากนั้นหมุนหัวไหล่ให้อยู่ในแนวเฉียง แล้ว คลอดไหล่หน้า
3. Woods corkscrew maneuver → ดันไหล่หลังไปอยู่แทนไหล่หน้าโดยดันจาก ด้านหน้าของไหล่หลัง หมุนทวนเข็ม 180 องศา
4. Rubin’s maneuver → ดันจากด้านหลังของไหล่ให้พับมาด้าน หน้า ให้ระยะห่างของไหล่ 2 ข้างลดลง&#x20;
5. Zavanelli maneuver → ดันเด็กเข้าไปในช่องคลอด + ดันค้างไว้จนกว่าจะผ่าตัดคลอด
