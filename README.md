# Med Llyr Survival Guide

## 📚 Welcome to MedLlyr Extern Survival Guide

ประกาศจากทีมวิชาการ Med Llyr (Med Llyr Academic Team)

ยินดีต้อนรับน้อง ๆ เข้าสู่ MedLlyr Extern Survival Guide คู่มือที่ตั้งใจรวบรวมขึ้นเพื่อช่วยให้น้อง ๆ สามารถรับมือกับการทำงานในฐานะ Extern ประเมินผู้ป่วยเบื้องต้นได้อย่างเป็นระบบ และเอาตัวรอดบนหอผู้ป่วยได้อย่างมั่นใจครับ

> 🚧 Work in Progress (อยู่ระหว่างการดำเนินการ)
>
> ขณะนี้ทางทีมวิชาการกำลังเร่งเรียบเรียงเนื้อหา ปรับปรุงข้อมูลให้ทันสมัย และตรวจสอบความถูกต้องทางการแพทย์อย่างละเอียด เนื้อหาบางส่วนในขณะนี้จึงอาจจะ <mark style="background-color:$danger;">**ยังไม่ครบถ้วนสมบูรณ์**</mark>
>
> ทั้งนี้ น้อง ๆ สามารถตรวจสอบความคืบหน้าและติดตามสถานะการอัปเดตของเนื้อหาในแต่ละ Part ได้โดยตรงจากตารางด้านล่างนี้ครับ
